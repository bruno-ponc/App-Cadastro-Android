# CadastroApp

Aplicativo Android desenvolvido em **Android Studio** para cadastro simples de informações pessoais.  
O app possui três telas principais: **Cadastro**, **Resultado** e **Sobre**.

---

## Funcionalidades
- Tela de **Cadastro**: permite inserir nome, idade e endereço.
- Tela de **Resultado**: exibe os dados cadastrados e confirma se estão corretos.
- Tela de **Sobre**: mostra informações da instituição, curso e integrantes da equipe.

---

## Requisitos
- Android Studio (versão recomendada: 2022.3 ou superior).
- Gradle configurado automaticamente pelo Android Studio.
- Versão mínima do Android: API 21 (Android 5.0 Lollipop).
